\# AI Suspicious URL Detector



\## Project Overview



This project uses Machine Learning to identify whether a URL is safe or suspicious.



Phishing websites often contain suspicious patterns like:

\- Long URLs

\- @ symbols

\- Multiple dots

\- HTTP usage

\- Suspicious keywords



The model learns these patterns and predicts the URL category.



\## Technologies Used



\- Python

\- Pandas

\- Scikit-learn

\- Machine Learning



\## ML Algorithm



Logistic Regression



\## Features Used



\- URL length

\- Presence of @ symbol

\- Number of dots

\- HTTP usage

\- Suspicious keywords



\## Output



The model predicts:



\- Safe URL

\- Suspicious URL



\## How to Run



Install libraries:



pip install -r requirements.txt



Run:



python url\_detector.py
